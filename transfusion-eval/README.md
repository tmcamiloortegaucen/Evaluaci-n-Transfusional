# Guía de Instalación y Uso

Esta aplicación es una página web estática compuesta por tres archivos:
- `index.html` (La estructura)
- `style.css` (El diseño)
- `script.js` (La lógica)

## ¿Cómo obtener un enlace para estudiantes?

Actualmente, estos archivos están solo en tu computadora. Para que tus estudiantes puedan acceder desde sus dispositivos, necesitas "hospedar" la app en internet.

### Opción 1: Netlify Drop (La más rápida)
1. Entra a [https://app.netlify.com/drop](https://app.netlify.com/drop).
2. Abre la carpeta donde están estos archivos en tu computadora.
3. Arrastra la carpeta completa al recuadro en la página de Netlify.
4. ¡Listo! Te darán un enlace (link) único que puedes copiar y enviar a tus alumnos.

### Opción 2: GitHub Pages (Más permanente)
Si tienes una cuenta de GitHub:
1. Crea un nuevo repositorio.
2. Sube estos tres archivos.
3. Ve a `Settings` > `Pages` y habilita la publicación desde la rama `main`.

## Edición
Si necesitas cambiar las reglas médicas, edita el archivo `script.js`.
Si quieres cambiar el texto visual, edita `index.html`.
